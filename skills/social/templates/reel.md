# Template: Reel (Instagram)

## Script de video

```
HOOK (0-3 seg):
[Frase que detenga el scroll. Dato sorprendente o pregunta provocadora.]

CONTEXTO (3-10 seg):
[Explica la situacion o el problema]

CONTENIDO (10-40 seg):
[El valor principal. 2-3 puntos clave.]
- Punto 1: [dato o insight]
- Punto 2: [dato o insight]
- Punto 3: [dato o insight]

CIERRE (40-50 seg):
[Conclusion + CTA]

TEXTO EN PANTALLA:
- [Texto principal que aparece]
- [Dato clave destacado]
```

## Caption del reel

```
[HOOK repetido o complementario]

Idea para video: muestra el resultado primero, luego el proceso.

[CTA segun objetivo]

[HASHTAGS — 5-8]
```

## Reglas para reels

- Duracion ideal: 30-60 segundos
- Hook en los primeros 3 segundos
- Texto en pantalla siempre (mucha gente ve sin sonido)
- Musica trending cuando aplique
