# Template: Post (Instagram/Facebook)

## Instagram

```
[HOOK — 1 linea que capture atencion]

[CUERPO — 3-5 parrafos cortos con el contenido principal]

[CTA — llamada a la accion segun objetivo]

[HASHTAGS — 5-10 relevantes]
```

## Facebook

```
[HOOK — misma linea que Instagram]

[CUERPO — puede ser mas largo y detallado que Instagram]

[CTA — misma llamada a la accion]

Contexto: [objetivo]. Formato: post.
```

## Diferencias entre plataformas

| Aspecto | Instagram | Facebook |
|---------|-----------|----------|
| Largo ideal | 150-300 palabras | 200-500 palabras |
| Hashtags | 5-10 al final | 3-5 al final |
| Emojis | Moderado | Minimo |
| Links | No (poner en bio) | Si, incluir link |
| Formato | Parrafos con saltos de linea | Mas fluido |
