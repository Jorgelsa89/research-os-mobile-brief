# CTA Library — Llamadas a la accion

CTAs organizados por objetivo del post.

## Por objetivo

| Objetivo | CTA |
|----------|-----|
| Marca personal | "Si esto te interesa, sigueme porque voy a documentar el proceso." |
| Ventas | "Si quieres algo parecido para tu negocio, escribeme." |
| Educacion | "Guardalo para revisarlo con calma." |
| Comunidad | "Que opinas? Te leo." |
| Actualizacion rapida | "Voy a seguir compartiendo avances." |

## CTAs adicionales por contexto

| Contexto | CTA |
|----------|-----|
| Research financiero | "Esto no es consejo financiero, pero si te interesa el analisis, sigueme." |
| Tutorial/Como hacer | "Guarda este post. Lo vas a necesitar." |
| Resultado personal | "El proceso importa mas que el resultado. Sigueme para ver como." |
| Pregunta abierta | "Dejame en comentarios que piensas." |
| Serie de posts | "Este es el post [X] de [Y]. Sigueme para no perderte los siguientes." |

## Reglas

- Un solo CTA por post (no confundir al lector)
- El CTA va al final, separado por una linea en blanco
- Tono conversacional, no agresivo
- No usar "dale like" o "comparte" (algoritmo lo penaliza)
