# Skill: Social (secundario)

## Que hago

Genero contenido optimizado para Instagram y Facebook. Adapto formato (post, reel, carrusel) al objetivo (marca personal, ventas, educacion, comunidad, actualizacion). Puedo crear contenido desde cero o basado en investigaciones existentes del vault.

## Cuando activarme

- "genera post" / "crea un post"
- "crea carrusel" / "contenido para Instagram"
- "publica sobre [tema]"
- "script para reel"
- "adapta [research] para redes"

## Output esperado

Archivo en `brain/knowledge/social/borradores/[YYYY-MM-DD]-[Tema]-[Formato].md` con:
- Frontmatter YAML
- Contenido para Instagram
- Contenido para Facebook (adaptado)
- Hashtags
- CTA

Cuando se confirma publicacion, mover a `brain/knowledge/social/publicados/`.

## Proceso

1. Recibir idea o identificar tema de `brain/knowledge/research/`
2. Determinar formato y objetivo
3. Aplicar brand voice
4. Usar template del formato correspondiente
5. Generar CTA segun objetivo
6. Generar hashtags
7. Crear version Instagram y version Facebook
8. Guardar como borrador en vault
9. Mostrar preview para aprobacion
