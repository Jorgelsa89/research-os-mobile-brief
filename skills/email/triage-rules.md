# Triage Rules — Clasificacion de correos

## Matriz urgencia/importancia

|  | Alta importancia | Baja importancia |
|--|------------------|------------------|
| **Urgente** | ACCION INMEDIATA — Responder hoy | RAPIDO — Respuesta corta o delegar |
| **No urgente** | PLANIFICAR — Agendar tiempo para responder | ARCHIVAR — Leer si hay tiempo, no requiere accion |

## Categorias

| Categoria | Icono | Accion |
|-----------|-------|--------|
| Accion requerida | ! | Necesita respuesta o accion de Jorge |
| Informativo | i | Solo lectura, no requiere accion |
| Seguimiento | -> | Jorge espera respuesta de alguien |
| Spam/Irrelevante | x | Ignorar o eliminar |
| Financiero | $ | Relacionado con trading, inversiones, o negocio |

## Formato de resumen

```
[!] De: [nombre] — [asunto]
    Resumen: [1-2 oraciones]
    Accion sugerida: [que hacer]
    Prioridad: [alta/media/baja]
```

## Reglas de priorizacion

1. Correos de personas conocidas > desconocidos
2. Financieros/negocios > informativos
3. Con deadline > sin deadline
4. Primera respuesta > cadena larga
