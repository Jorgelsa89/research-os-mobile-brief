# End of Day Template

Estructura del cierre del dia. Se agrega a la nota diaria existente.

```markdown
## Cierre del dia

### Que se hizo hoy

- Investigaciones: [N] ([lista con links])
- Posts generados: [N]
- Correos procesados: [N]
- Watchlist actualizada: [si/no]

### Metricas

| Metrica | Valor |
|---------|-------|
| Notas creadas | [N] |
| Notas modificadas | [N] |
| Skills ejecutados | [lista] |

### Pendientes para manana

- [ ] [Pendiente 1]
- [ ] [Pendiente 2]
- [ ] [Pendiente 3]

### Nota personal

[Observacion opcional sobre el dia — algo aprendido, algo a mejorar]
```
