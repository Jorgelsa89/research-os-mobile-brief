# Registro de sesiones

Cada sesion de trabajo con cualquier AI se registra aqui para mantener
continuidad entre sesiones y proyectos.

## Formato

```
### [FECHA] — [PROYECTO]
- Duracion: [estimada]
- Skills usados: [lista]
- Que se hizo: [resumen en 1-3 lineas]
- Que quedo pendiente: [lista corta]
- Aprendizaje: [si hubo algo nuevo]
```
