## Flujo tipico detectado

1. Ve algo interesante (noticia, idea, screenshot)
2. Pregunta si es viable / como se haria
3. Si le convence, dice "hagamoslo"
4. Prefiere que el sistema construya y el revise
5. Da feedback y ajusta

## Sesiones de trabajo

- Horario tipico: (pendiente de detectar con mas sesiones)
- Duracion tipica: (pendiente)
- Desde que dispositivo: movil + escritorio

## Como se actualiza

El sistema observa:
- A que hora se conecta Jorge
- Cuanto duran las sesiones
- Que tipo de tareas hace primero
- Cuando deja tareas para despues
