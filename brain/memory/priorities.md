# Sistema de prioridades — La amigdala digital

Esto define que le importa a Jorge y en que orden. El sistema usa estas
prioridades para decidir que mostrar primero, que sugerir, y como ordenar
la informacion.

## Prioridades por dominio (orden de importancia)

1. **Seguridad financiera** — Proteger capital, gestionar riesgo
2. **Crecimiento de conocimiento** — Aprender, investigar, entender
3. **Generacion de ingresos** — Trading, negocios, oportunidades
4. **Productividad personal** — Automatizar, optimizar tiempo
5. **Presencia digital** — Contenido, marca personal

## Reglas de priorizacion

- Si hay conflicto entre dominios, el de mayor prioridad gana
- Alertas financieras siempre aparecen primero en el brief diario
- Investigaciones con score > 8.0 se marcan como urgentes
- Correos de alta prioridad se mencionan antes que borradores sociales

## Valores fundamentales

- **No inventar datos** — Prefiere "no se" a informacion falsa
- **Seguridad primero** — Nunca exponer datos sensibles
- **Accion sobre teoria** — Prefiere una accion concreta a un analisis largo
- **Simplicidad** — La solucion mas simple que funcione
- **Autocuidado** — Valora el cuidado personal como parte integral de la vida
- **Familia y relaciones** — Gran esposo, amigo y apoyo familiar
- **Curiosidad** — Explorar, aprender, cuestionar supuestos constantemente
- **Independencia** — Multiples fuentes de ingreso, no depender de una sola

## Como se actualiza

Cuando Jorge toma decisiones repetidas que indican un cambio de prioridades,
el sistema propone actualizar este archivo.
